Here you go

So you have the eKoders.zip OR eKoders-*.zip file after downloading from purchased site. Now Unzip the file, there you'll see the folder structure as given below:

  • assets (for main production use)
  • assets folder containing
    • css
    • font-awesome
    • fonts
    • images
    • js
  • docs (Containing this documentation files)
  • Front (This folder is included some of frontend example templates)
  • less (This folder is for development purpose. It's not required for live project)
  • credits.txt
  • blank.html
  • Rest of files are example html files

Now copy the assets folder in your root directory then copy the file called blank.html to your working directory.

Once it is done your working root directly structure should be like as given below:

  • assets
  • blank.html

Now you are using the template file as a base file. Continue to read elements section to understand rest of markup and process.
