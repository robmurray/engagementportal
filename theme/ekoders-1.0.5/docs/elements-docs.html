Tabs

eKoders admin template uses the same tabs element and classes of Bootstrap. For eKoders style tabs your tab codes should be between an additional DIV with class .tc-tabs. For example please see the code given bellow:

<div class="tc-tabs">

	<!-- Nav tabs -->
	<ul class="nav nav-tabs">
		<li class="active"><a href="#home" data-toggle="tab">Home</a></li>
		<li><a href="#profile" data-toggle="tab">Profile</a></li>
		<li><a href="#messages" data-toggle="tab">Messages</a></li>
		<li><a href="#settings" data-toggle="tab">Settings</a></li>
	</ul>
	<!-- Tab panes -->
	<div class="tab-content">
		<div class="tab-pane active" id="home">...</div>
		<div class="tab-pane" id="profile">...</div>
		<div class="tab-pane" id="messages">...</div>
		<div class="tab-pane" id="settings">...</div>
	</div>
	
</div>

For example

<ul class="nav nav-tabs tab-color-dark background-dark">

For example

CSS3 Checkbox, Radio and On/Off Switches

eKoders comes with pure CSS3 styled checkbox without any javascript. Switches are also animated on supported browsers such as Firefox and Chrome. It also falls back to default browser style on IE8.

The following exta options are defined for checkbox and radios for .tc element:

.tc-success.tc-danger.tc-info.tc-primary.tc-warning

For Examples

<div class="tcb"><!--Its replaced by defualt bootstrap checkbox DIV class, must require in form-->
  <label>
	<input type="checkbox" class="tc">
	<span class="labels" for="checkbox"> Check me</span>
  </label>
</div>

<label>
  <input type="checkbox" class="tc">
  <span class="labels" for="checkbox"> Check me</span>
</label>

Switch Samles

<label>
  <input name="switch-field-1" class="tc tc-switch" type="checkbox" />
  <span class="labels"></span>
 </label>

<label>
  <input name="switch-field-1" class="tc tc-switch tc-switch-3" type="checkbox" />
  <span class="labels"></span>
</label>

Currently switches come in 7 styles: .tc-switch , .tc-switch.tc-switch-2, ... .tc-switch-.tc-switch-7

Note You must include an element with .labels class right after the checkbox ,radio or swicth. It may or may not contain text.

A message item in the list consists of different elements:

  1. A checkbox to select the message
  2. .message-star A star icon to mark messages as starred/unstarred, Must be use Fontawsoe icon fa fa-star-o like
    3. <a href="#"><i class="message-star fa fa-star-o"></i></a>
  3. .sender Which is the sender's name
  4. .time Which is the time the message was sent/received
  5. .attachment Which indicates there's an attachment
  6. .summary Which contains message summary wrapped inside a .text and an optional .mail-tag or .message-flags
