Settings Layout Options

ekoders layout settings use localStorage to saved setting inside assets/js/main.js By defualt its set Fixed Navbar and Fixed Sidebar, If you want custom layout option please follow bellow instructions:

Fixed Navbar

For this you should add .fixed class in .navbar-topelement

<nav class="navbar-top fixed" role="navigation">
	........
</nav>

Inside Container

For Inside container (fixed width), you should add .container class to #main-container and .navbar-top elements:

<div id="main-container" class="container">
	<nav class="navbar-top container" role="navigation">
		......
	</nav>
</div>

Sidebar Settings

For Fixed Sidebar you should add .fixed class to following bellow elements

.navbar-top .navbar-side .breadcrumbs #page-wrapper

For Collapsed Sidebar you should add .collapsed class to following bellow elements

.navbar-side #page-wrapper .breadcrumbs .footer-inner

For Light Color Sidebar just add .sidebar-light in .navbar-side element
