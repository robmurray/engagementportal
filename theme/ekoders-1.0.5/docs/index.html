Upgrade Instruction 1.0.5 (Latest) and Change logs
  1. For upgrade, replace the files under /assets/css/themes/ or rebuild by using "Less"

  2. Toogle function added in ../assets/js/main.js line 176-185 and 187-210 (for fix page scrollbars issue in modals) OR just replace this file if there are no modifications, By this you can use .toggle class to make toggle button anywhere whenever you required, For example, in "inobx.html" page we have used cc and bcc options for composing and replying an email.

  3. If your modal has long contents you should use class .modal-scroll in modal div element along with class .modal.

  4. Replace folders ../assets/css/plugins/jqueryui/ and ../assets/js/plugins/jqueryui/, jQuery UI plugin files have been renamed and upgraded by latest version, Removed version name for easy upgrade, now you can include them like ../assets/css/plugins/jquery-ui.min.css and ../assets/js/plugins/jquery-ui.min.js.

  5. Replace folder ../assets/css/plugins/select2/ and ../assets/js/plugins/select2/, Select2 plugin has been upgraded by latest version and now there is no need to use custom CSS, just include only single css../assets/css/plugins/select2/select2.css.

  6. Replace the folder ../assets/js/plugins/bootstrap-markdown/, Markdown editor plugin has been upgraded by latest version and you have to include ../assets/js/plugins/bootstrap-markdown/markdown.js before ../assets/js/plugins/bootstrap-markdown/bootstrap-markdown.js for more options see plugins page.

  7. Replace folder ../assets/js/plugins/datatables/ jQuery Data Tables plugin has been upgraded by Latest version and added few more advance datatable example which us used in eCommerce >> products.html, for more information please see plugins page and also ../assets/js/plugins/datatables/datatables.init.js.

  8. New Folders added in ../assets/js/plugins/
    1. jquery-nestable
    2. bootstrap-wysiwyg
    3. colorpickers
    New Plugins added Nestable lists, Bootstrap Wysiwyg Editor, Color Picker and Custom Color Picker, Please see plugins page to know how to use.

  9. Additional wrapper added for X-editable ../assets/js/plugins/bootstrap-editable/ek-editable.js, to use more options in X-editable Inline Editable, please see plugins page.
