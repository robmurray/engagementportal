Features

eKoders is a lightweight and feature rich admin template which is clean and easy to use and based on bootsrap 3.x.
Current release comes with the following features

  • Pure CSS3 checkbox, radio and On/Off switch elements (not available in IE8)
  • Portlets in different colors and styles
  • Tiles buttons in different colors and styles
  • Responsive jQuery datatable.
  • 5 Color Styles
  • 3 additioanl Buttons styles
  • Docs type additioanl note.
  • Mulit styled tabs.
  • Three style accordions.
  • Custom layouts
Dropdown, progress bars, jQuery Sliders, buttons, checkboxes, radios, labels, badges, Gritter Notifications, tooltips and text can be used in multi color classes with markup. The example codes are given in further pages.

You can add more colors by using variables.less. If you are not familiar with less please share your desired color scheme by mail, I'd be happy to make for you
